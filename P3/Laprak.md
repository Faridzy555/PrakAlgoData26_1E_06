### Jawaban Percobaan 1:
1. Tidak harus karena:
- Class boleh hanya punya atribut saja.
- Class juga boleh hanya punya method saja.
- Array of object hanya menyimpan objek dari class tersebut.
2. Membuat array berisi object Mahasiswa sebanyak 3 elemen.
3. Iya, mahasiswa memiliki konstruktor, bisa di lakukan pemanggilan karena java akan otomatis memberikan construktor default jika belum ada.
4. Kode ini digunakan untuk memasukkan data mahasiswa ke posisi pertama di dalam array.
5. Mengapa class Mahasiswa dan MahasiswaDemo dipisahkan?

Karena prinsip OOP (Object Oriented Programming):

Mahasiswa: model/data

MahasiswaDemo: program utama

Keuntungan dipisah:
- kode lebih rapi
- mudah digunakan ulang
- mudah maintenance
- sesuai konsep modular programming
